# 用户全生命周期测试用例

> 创建日期: 2026-02-16
> 状态: Active
> 版本: 1.0.0

## 概述

本文档定义了 CodeCoder 四种用户类型的全生命周期测试用例，确保所有功能被完整覆盖。

## 用户类型定义

| 类型 | 标识符 | 核心 Agent | 核心场景 |
|------|--------|-----------|---------|
| 软件开发者 | ULC-DEV | build, plan, code-reviewer, security-reviewer, tdd-guide, architect, explore | 代码开发、质量审查、架构设计 |
| 内容创作者 | ULC-CRT | writer, proofreader, expander, expander-fiction, expander-nonfiction | 长文写作、内容校对、书籍扩展 |
| 决策分析师 | ULC-ANL | observer, decision, macro, trader, picker, miniproduct, ai-engineer | 宏观分析、交易决策、产品选品 |
| 逆向工程师 | ULC-REV | code-reverse, jar-code-reverse | 网站逆向、JAR包分析 |
