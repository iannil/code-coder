## 一、书籍 (Books)

共 5 卷 26 本书，基于"进阶 rc"哲学框架，探讨从宇宙本体到个体实践的完整思想体系。

### 核心概念

- **可能性基底**：宇宙的源头是包含无限潜能的动态场域，而非静态实体
- **观察与收敛**：通过"观察"行为，可能性被收敛为确定的现实
- **可用余量**：尚未被固化的潜能空间，是自由意志和创造力的来源
- **评估异化**：评估能力被外部系统剥夺，导致主体性丧失

## 三、产品 (Products)

共 16 个开源技术产品，覆盖 AI 基础设施、无人机、数据同步等领域。

### AI 基础设施

| 产品 | 技术栈 | 核心功能 |
|------|--------|----------|
| **Code-Coder** | Bun + TypeScript + SolidJS | AI 个人工作台，23+ 专业 Agent，30+ AI 提供商支持，CLOSE 决策框架 |
| **Ascend Docker Kit (ADK)** | Python | 华为昇腾 NPU 环境 DevOps 工具，解决 CANN/驱动/框架版本兼容性 |
| **GPU Dead Node Detector (GDND)** | Rust + Kubernetes | GPU 健康监控与故障隔离，三级检测流水线，自动 Cordon/Taint |
| **PII Airlock** | Python + FastAPI | LLM 调用中的 PII 脱敏中间件，支持流式响应和模糊匹配恢复 |
| **One Data Studio (ODS)** | React + Python | DataOps + MLOps + LLMOps 融合平台，整合 Alldata/Cube/Bisheng |
| **Hetero-Compute-Router (HCR)** | - | 异构计算路由器 |
