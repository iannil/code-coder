在我们探寻宏观经济的壮丽图景时，有一个领域，它或许不像GDP那样宏大，不像投资那样轰轰烈烈，也不像金融数据那样瞬息万变，但它却是一切经济活动的基石与归宿。这个领域，就是就业与收入。

在我看来，就业与收入，是连接宏观经济与微观个体的“脐带”。宏观经济的每一次冷暖变化，最终都会通过这条“脐带”，转化为亿万家庭餐桌上的菜肴、孩子们身上的新衣，以及每个人对未来的憧憬或忧虑。

在上一章，我们探讨了消费。但消费不是无源之水、无本之木。它的源头活水，正是来自于稳定而有增长的就业与收入。一个没有安全感的工人，一个收入朝不保夕的白领，一个对未来感到迷茫的大学毕业生，他们不可能成为消费市场的中坚力量。因此，对就业与收入的分析，是所有消费分析的“前传”，是理解和预测内需动能的最根本的出发点。

就业问题，在中国这样一个拥有14亿人口、每年有上千万新增劳动力的国家，其重要性无论如何强调都不为过。它不仅是一个经济问题，更是一个社会问题、政治问题。“稳就业”，始终是宏观调控政策篮子中最优先的目标之一。

然而，就业与收入的数据，也常常是争议与困惑的焦点。“官方公布的失业率那么低，为什么我感觉身边找工作很难？”、“居民收入又增长了，为什么我的工资没涨？”……这些疑问，反映了宏观数据与微观感受之间的巨大鸿沟。

在本章，我们将直面这些挑战。我们将像一位社会学家与经济学家的结合体，去深入探究中国就业与收入的真实图景。我们会系统梳理失业率、新增就业等核心指标，并毫不避讳地探讨这些数据背后的“虚”与“实”。我们还将解剖居民收入的来源、增长与分配结构，洞察共同富裕背景下的新变化。最重要的是，我们将再次聚焦于“就业-收入-消费”的传导链，将其作为一个完整的分析框架，来审视中国内需的根基是否稳固。

读懂了就业与收入，你才能真正理解中国经济的底色，感知社会的脉动，并对消费的未来，做出最深刻的洞察。

## 9.1 核心就业指标概览：失业率、新增就业

要评估一个国家的就业状况，我们需要一套“体检指标”。中国的就业统计体系，近年来不断完善，形成了一系列相互补充的指标。我们首先要熟悉这些指标的定义和意义。

### 城镇调查失业率：就业市场的“温度计”

这是当前衡量就业状况最核心、最权威的月度指标。

* 定义： 指通过劳动力调查（抽样调查）获取的，城镇常住人口中，失业人口占劳动力人口（就业人口+失业人口）的比重。
* 关键定义辨析：
  * 劳动力人口： 指16周岁及以上，有劳动能力，在调查周内，为取得劳动报酬或经营收入而工作了至少1小时的人（就业人口），以及正在寻找工作的人（失业人口）。
  * 失业人口的三个标准（国际劳工组织标准）： 必须同时满足以下三个条件，才算失业人口：
        1. 在调查周前，没有工作（工作小于1小时）。
        2. 在最近3个月内，采取了具体行动寻找工作（如投简历、参加招聘会、托人介绍等）。
        3. 一旦有合适的工作，在2周内能够开始工作。
  * 非劳动力人口： 16周岁及以上，但没有工作，也不寻找工作的人。比如，在校学生、家庭主妇、退休人员、丧失劳动能力者，以及那些想工作但因为种种原因暂时放弃寻找工作的人（“丧失信心的工人”）。这些人，不计入失业率的分母和分子。

* 数据的发布与构成：
  * 国家统计局每月公布全国城镇调查失业率。
  * 同时会公布两个重要的结构性数据：
    * 31个大城市城镇调查失业率： 反映大城市就业状况，通常对经济周期变化更敏感。
    * 16-24岁青年调查失业率： 这是衡量青年群体（主要是高校毕业生）就业压力的关键指标，近年来受到社会高度关注。

* 分析价值：
    城镇调查失业率，是宏观调控的重要参考。当它持续攀升，并突破政府设定的目标（如5.5%）时，通常会触发更大力度的“稳就业、稳增长”政策。它也是观察经济周期的同步或略滞后指标，经济下行，企业裁员，失业率上升。

### 城镇新增就业人数：就业的“创造能力”

* 定义： 指在报告期内，城镇各类单位（包括私营企业、个体户）新招用的、初次就业和再就业的人员总数。
* 来源： 这个数据主要来自于人力资源和社会保障部的行政记录（如就业登记、社保缴纳记录），而非抽样调查。
* 分析价值：
  * 衡量就业创造： 它反映了经济体在一段时间内，创造了多少新的就业岗位。每年“两会”上，《政府工作报告》都会设定一个“城镇新增就业”的目标（如1200万人以上），这是政府的核心KPI之一。
  * 与失业率互为补充： 新增就业，衡量的是就业的“流入”；而失业，则更多地反映了就业的“流出”（失业）和存量的“摩擦”。两者结合，才能更全面地看问题。即使新增就业完成了目标，但如果因为产业结构调整、企业倒闭等原因，导致失业人数更多，总的就业压力依然可能很大。

### 农民工监测调查数据：流动的“蓄水池”

农民工，是中国劳动力市场一个规模庞大且极具特色的群体。他们像候鸟一样，在城乡之间流动，成为中国经济的“弹性”所在和就业的“蓄水池”。

* 核心指标（季度发布）：
  * 农民工总量：
  * 外出农民工和本地农民工的数量。
  * 外出农民工月均收入。
* 分析价值：
  * 经济的“晴雨表”： 农民工主要集中在制造业和建筑业等周期性强的行业。当经济景气时，工厂订单多，工地繁忙，大量农民工进城务工，外出农民工数量增加。当经济下行时，企业裁员、工地停工，农民工是首先受到冲击的群体，他们中的许多人会选择返乡，导致外出农民工数量减少。
  * 理解“隐性失业”： 许多返乡的农民工，虽然在城市里失去了工作，但在统计上可能因为回到农村从事农业生产（或无业但不再寻找城镇工作），而不被计入城镇调查失业率。因此，观察外出农民工数量的变化，是理解中国真实就业压力，特别是“隐性失业”问题的一个重要窗口。如果城镇调查失业率看似稳定，但外出农民工数量却在大幅减少，这同样是一个需要高度警惕的信号。

### PMI从业人员指数：企业的“用人意愿”

* 定义： 采购经理指数（PMI）中的一个分项，通过询问企业“你们本月的用工人数相比上月是增加、持平还是减少了？”来编制。
* 分析价值：
  * 领先指标： PMI是月度最高频的领先指标之一。从业人员指数的变化，预示着未来一到两个月就业市场的冷暖。
  * 企业视角的印证： 它从企业的角度，为官方的失业率数据提供了一个交叉验证。如果PMI从业人员指数持续低于50的荣枯线，说明企业用人意愿偏弱，即使失业率数据暂时平稳，未来的就业压力也在积聚。

通过综合运用这四大类指标，我们就能从总量与结构、城镇与乡村、宏观与微观等多个角度，对中国的就业状况，形成一个相对立体和全面的认知。

## 9.2 失业率数据的“虚”与“实”

“5.2%的失业率，意味着100个想工作的人里，只有5个多一点没找到工作。这和我的感觉完全不一样啊！”——这可能是公众对就业数据最大的困惑。

作为一个专业的分析师，我们不能简单地用“感觉”来否定数据，也不能盲目地相信数据而忽视现实。我们的责任，是去理解数据与“感觉”之间产生鸿沟的原因。这个鸿沟，既来自于统计口径的“虚”，也来自于中国就业市场的“实”。

### 统计口径的“虚”：失业率为何总是“看起来很美”？

城镇调查失业率数据本身是科学、严谨的，符合国际标准。但正是这个“国际标准”，在中国的具体国情下，可能会导致一些信息的“失真”。

* “失业”的严格定义，排除了一大批人：
    回想一下失业人口的三个标准：“没工作”、“在找工作”、“能马上工作”。
  * “丧失信心的工人”被排除： 许多人，特别是在经济长期不景气时，可能因为找不到合适的工作而暂时放弃了寻找（比如，找不到专业对口工作的大学生，选择回家“躺平”备考）。他们在主观上是想工作的，但因为没有“在找工作”这个行为，就不被统计为失业人口，而是被划为“非劳动力”。这部分人越多，失业率被低估得就越严重。
  * 每周工作1小时就算“就业”： 这个标准非常低。一个每周只打几个小时零工的人，其收入可能远不足以维持生计，生活状态与失业无异。但在统计上，他被算作“就业人口”。这种情况，我们称之为“不充分就业”。调查失业率无法反映“不充分就业”的问题。

* 统计范围仅限“城镇”，忽略了城乡二元结构：
  * 返乡农民工的“隐性失业”： 这是中国最特殊的国情。一个在城市失业的农民工，如果回到农村，他就不再是城镇常住人口，自动退出了城镇调查失业率的统计样本。即使他在农村无事可做，也只是增加了农村的隐性失业，而不会推高城镇失业率。这种城乡二元的“防火墙”机制，使得城镇调查失业率在反映由周期性冲击（如出口下滑、地产下行）导致的整体就业压力时，存在天然的“钝化”。这就是为什么我们必须结合“外出农民工数量”来分析的原因。

### 现实世界的“实”：我们感受到的“就业难”难在哪里？

公众感受到的“就业难”，与统计数据之间的差异，还源于现实中就业问题的结构性、摩擦性特征，这些是单一的失业率数字无法完全展现的。

* 结构性失业： “有活没人干，有人没活干”
  * 产业转型带来的错配： 随着中国经济向高技术、高附加值方向转型，对高技能人才的需求在增加，而传统制造业、低端服务业的岗位在减少。这导致，一方面企业招不到合适的工程师、技术工人；另一方面，大量低技能劳动者和专业不对口的大学毕业生找不到工作。
  * 青年就业的“结构性矛盾”尤为突出： 每年上千万的高校毕业生涌入市场，但他们期望的白领岗位增长有限，而他们又不愿进入制造业或基层服务业岗位。这种“期望”与“现实”的错配，是青年失业率高企的核心原因之一。
* 摩擦性失业： 换工作期间的“空窗期”
    即便是健康的劳动力市场，也总会存在一部分人处于换工作的“空窗期”，这是正常的摩擦。但如果经济环境不好，求职过程变长，这种摩擦性失业的周期也会被拉长。
* 就业质量的下降： “有工作，但不是好工作”
    很多人感受到的“难”，不一定是完全找不到工作，而是找不到“体面”的、“稳定”的、“有发展”的工作。
  * 灵活性就业的兴起： 平台经济的发展，创造了大量外卖骑手、网约车司机等岗位。这些岗位虽然解决了部分人的“吃饭”问题，但往往存在工作不稳定、社会保障缺失、职业发展空间有限等问题。一个名校毕业生去送外卖，虽然在统计上是“就业”了，但在社会观感和个人发展上，无疑是一种“向下兼容”。
  * “内卷”与工作强度的增加： 经济不景气时，企业为了控制成本，可能会要求员工“一人干三人的活”，导致在职人员的工作强度和压力剧增。这也是“就业难”的一种现实体现。

分析师的“修正”之道：
面对失业率的“虚”与“实”，一个专业的分析师绝不会只盯着那个5.2%的数字。他会：

1. 看结构，而非总量： 青年失业率的变化，比总体失业率更能反映经济的“痛感”。
2. 看互证，而非孤证： 将失业率与农民工数据、PMI从业人员指数、乃至网络招聘平台的职位发布数量等指标，进行交叉验证。
3. 看质量，而非数量： 关注居民收入增长情况、工作时长的变化、社会保险覆盖面的扩大情况，来间接评估就业质量。
4. 定性与定量结合： 在分析报告中，除了引用官方失业率，更要用文字去描述和分析结构性失业、不充分就业等“数字背后”的真实问题。

只有这样，我们才能提供一份既尊重数据、又贴近现实的、负责任的就业分析。

## 9.3 居民收入分析：来源、增长与分配

就业是“因”，收入是“果”。居民的“钱袋子”有多鼓，直接决定了消费的“底气”有多足。

### 居民可支配收入：我们真正能花的钱

* 定义： 指居民可用于最终消费支出和储蓄的总和。
    可支配收入 = 总收入 - 个人所得税 - 个人缴纳的社会保险费
* 名义增速 vs. 实际增速：
    和社零一样，我们更关心的是剔除了通胀影响的实际增速。
    居民收入实际增速 = 居民收入名义增速 - CPI
    只有当居民收入的实际增速为正，并且最好是快于GDP增速时，我们才能说居民的“蛋糕”在整个国民经济的“大蛋糕”中切分到的份额在增加，这是实现“共同富裕”和“消费驱动”的宏观基础。

### 收入的四大来源：钱从哪里来？

国家统计局将居民可支配收入分解为四个部分，分析这个结构，能让我们看清收入增长的动力源。

1. 工资性收入（占比约55-60%）：
    * 定义： 居民因从事主要职业、第二职业以及打零工等活动所取得的劳动报酬。这是绝大多数居民收入的绝对大头。
    * 驱动因素： 直接与就业状况和企业效益挂钩。经济上行，企业利润好，就业稳定，工资和奖金自然水涨船高。反之亦然。因此，工资性收入的增速，与宏观经济周期高度同步。
2. 经营净收入（占比约15-20%）：
    * 定义： 居民从事生产经营活动所获得的净收入，如个体工商户、小作坊主、开设网店等。
    * 驱动因素： 反映了中小微企业和个体户的景气度。营商环境的改善、减税降费政策的落实，都会提振经营净收入。在经济下行期，这部分收入受到的冲击通常最大。
3. 财产净收入（占比约8-10%）：
    * 定义： 居民拥有的动产（如银行存款、有价证券）和不动产（如房屋、车辆）所获得的收入。主要包括利息、股息红利、以及房租收入。
    * 驱动因素：
        * 金融市场表现： 牛市能带来丰厚的股息和资本利得，熊市则会侵蚀财产性收入。
        * 房地产市场表现： 房价和租金的上涨，会增加房东的收入。
    * “财富效应”的来源： 财产净收入的增长，是“财富效应”的直接体现。资产价格的上涨，让人们感觉自己“变富了”，从而更敢于消费。这部分收入的顺周期性和波动性极强。
4. 转移净收入（占比约15-20%）：
    * 定义： 指国家、单位、社会团体对居民的各种转移支付，以及居民之间的收支相抵的净额。主要包括养老金、离退休金、社会救济和补助、报销医疗费、住户间的赡养收入等。
    * 驱动因素： 主要由社会保障和再分配政策决定。
    * “稳定器”与“逆周期”调节器： 转移净收入具有很强的稳定性。即使在经济下行、工资性收入下降时，养老金、失业救济金等仍能为居民提供一个基础的收入保障。在“稳增长”时期，政府还可以通过提高养老金标准、发放消费券（本质上是一种转移支付）等方式，来精准地托底中低收入群体的收入和消费。

### 收入分配：K型分化的挑战

总量数据背后，是巨大的结构性差异。分析收入，必须关注分配问题。

* 城乡差距： 城镇居民的人均可支配收入，长期显著高于农村居民。近年来，随着乡村振兴战略的推进和农民工收入的增长，城乡收入比有所收窄，但差距依然巨大。
* 阶层差距（更核心的挑战）：
  * 虽然官方不定期公布基尼系数，但从各种迹象来看，中国居民收入的“K型”分化现象是客观存在的。
  * 高收入群体： 其收入更多地来源于财产净收入和高薪酬的工资性收入。在资产价格上涨的周期中，他们的收入增长最快。
  * 中低收入群体： 其收入主要依赖于工资性收入和转移净收入。他们的收入增长，更多地取决于宏观经济的景气度和社会保障的完善程度。
  * 为何重要？ 如前所述，中低收入群体的边际消费倾向（MPC）远高于高收入群体。 这意味着，同样增加1元钱的收入，给一个低收入家庭带来的新增消费，要远远大于给一个高收入家庭。因此，任何一个旨在“促消费”的政策，如果不能有效地提升中低收入群体的收入，其效果都将大打折扣。 改善收入分配结构，推进共同富裕，不仅仅是一个社会公平问题，更是一个释放内需潜力的核心经济问题。

## 9.4 分析框架：就业、收入与消费的传导链

我们现在可以将本章和上一章的所有知识点，整合成一个更加完整、更加强大的“就业-收入-消费”黄金三角分析框架。这个框架，是研判中国内需基本盘的“总纲”。

传导链条：
`宏观经济/政策环境 → (1) 就业市场 → (2) 居民收入 → (3) 居民信心/储蓄行为 → (4) 最终消费`

分析步骤与关键问题：

### 第一步：分析就业市场（饭碗问题）

* 关键指标： 城镇调查失业率（总量+青年）、新增就业、PMI从业人员、农民工数量。
* 要回答的问题：
  * 就业市场的总体压力是在上升还是在下降？（看失业率趋势）
  * 结构性矛盾是否突出？（看青年失业率）
  * 企业的用人意愿是强还是弱？（看PMI）
  * 周期性冲击对低收入群体的影响有多大？（看农民工流动）
  * 是否存在大量的“隐性失业”和“不充分就业”？（定性判断）

### 第二步：分析居民收入（钱包问题）

* 关键指标： 居民人均可支配收入（总量+城乡结构）、收入的四项构成。
* 要回答的问题：
  * 居民收入的真实增速，能否跑赢GDP和CPI？
  * 收入增长的主要动力是什么？是健康的工资性收入增长，还是不稳定的财产性收入增长？
  * 中低收入群体的收入状况如何？（通过经营净收入、农民工收入等间接判断）
  * 再分配政策（转移净收入）是否起到了有效的“托底”作用？

### 第三步：分析信心与储蓄（意愿问题）

* 关键指标： 消费者信心指数、居民存款（储蓄率）。
* 要回答的问题：
  * 居民对未来的预期是乐观还是悲观？
  * “预防性储蓄”的倾向是在增强还是在减弱？
  * 是否存在“有钱不敢花”的现象？（即收入增长，但消费信心和消费增速不振）
  * 资产价格的变化（财富效应/负财富效应）对居民信心的影响如何？

### 第四步：预判最终消费（结果）

* 关键指标： 社零总额（名义+实际）、分项商品零售、服务消费（通过交通、旅游、电影票房等高频数据代理）。
* 形成综合判断：
  * 最理想的状态（全面复苏）： 就业改善 → 收入稳健增长（特别是工资性收入） → 信心恢复，储蓄率下降 → 消费（特别是可选消费）全面回暖。
  * 最糟糕的状态（衰退）： 失业率攀升 → 收入下降 → 信心崩溃，储蓄率飙升 → 消费（特别是可选消费）急剧萎缩。
  * 复杂的现实（结构性复苏/K型分化）： 就业市场结构性矛盾突出 → 收入增长分化，中低收入群体承压 → 信心修复缓慢，预防性储蓄高企 → 消费总量增长乏力，结构上“必需强于可选”、“线上强于线下”、“高端与性价比并存”。

通过这个四步走的分析框架，我们就能将对消费的分析，从简单的“看社零猜趋势”，深化为对经济社会运行根本逻辑的洞察。它能帮助我们解释过去，理解现在，并对未来，做出更有依据的判断。
