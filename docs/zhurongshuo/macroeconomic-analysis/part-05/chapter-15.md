在宏观经济的舞台上，如果说市场是一只“看不见的手”，在自发地配置资源；那么财政，就是那只“看得见的手”，在关键时刻，通过收入与支出，对经济进行着强有力的引导、调节和干预。这只手，直接掌握着政府的“钱袋子”，决定了修多少路、建多少学校、给企业减多少税、给民众发多少补贴。

在我四十年的研究生涯中，对财政数据的分析，是理解国家战略意图和政策力度的必经之路。与追求盈利的企业不同，政府的财政行为，其目标是多元的：稳增长、调结构、惠民生、防风险……每一笔财政收入的征收，每一笔财政支出的投向，都蕴含着深刻的政策考量。

财政政策，与货币政策并称为宏观调控的“两大支柱”。

* 货币政策像是在调节水的“总量”和“价格”（利率），它更偏向于总量调控，作用于金融市场。
* 财政政策则更像是精准的“滴灌”，它可以将资源直接、定点地输送到最需要的领域，作用于实体经济。减税降费，直接让利于企业和居民；基建投资，直接创造总需求；民生补贴，直接托底困难群体的生活。

因此，财政政策往往被认为在“调结构”和“稳增长”方面，更直接、更有效。特别是在经济下行、市场主体信心不足时（即“流动性陷阱”），单纯的货币“放水”可能失效，此时，积极的财政政策，就成为打破僵局、启动经济的关键力量。

然而，政府的“钱袋子”也并非取之不尽。财政收支的背后，是关于赤字、债务和可持续性的严肃考量。读懂财政数据，不仅要看政府“想做什么”，还要看它“能做什么”，以及“这样做的代价是什么”。

在本章，我们将像一位资深的预算审查官和国会预算办公室（CBO）的分析师，去全面审视中国的财政体系。我们会先厘清公共预算和政府性基金预算这两本最重要的“账本”。接着，我们将构建一个从财政收支的规模、节奏和结构，来判断政策力度与方向的分析框架。最后，我们将打通赤字、国债与市场流动性之间的联动，理解财政行为如何深刻地影响金融市场。

读懂财政，你才能真正读懂政府在想什么、在做什么，才能在纷繁复杂的政策信号中，找到最坚实的投资逻辑。

## 15.1 公共预算与政府性基金预算

要分析财政，我们首先要搞清楚政府的“钱”都放在哪些“口袋”里。中国的政府预算，由四本账构成：一般公共预算、政府性基金预算、国有资本经营预算、社会保险基金预算。其中，与我们宏观分析最密切相关的，是前两本——一般公共预算和政府性基金预算。它们共同构成了我们常说的“广义财政”。

### 一般公共预算：政府的“主账户”

这是政府最核心、最主要的“钱袋子”，反映了政府凭借国家权力（主要是税收）筹集的、用于保障公共服务和国家机器运转的收支活动。

* 收入来源：
    1. 税收收入（绝对大头，占比约85%）：
        * 增值税： 中国第一大税种，与工业生产、商业活动高度相关，是经济的“晴雨表”。
        * 企业所得税： 与企业利润直接挂钩，反映企业经营效益。
        * 个人所得税： 与居民收入和就业状况相关。
        * 消费税、关税、印花税等。
        税收收入的增速，是判断经济景气度和企业/居民部门活力的重要“硬指标”。
    2. 非税收入： 包括行政事业性收费、罚没收入、国有资源（资产）有偿使用收入等。

* 支出方向（按功能分类）：
    这是判断财政政策发力方向的关键。
  * 民生类支出（“刚性”支出）： 教育、社会保障和就业、卫生健康等。这部分支出占比高，且逐年增长，具有很强的刚性。
  * 经济建设类支出： 城乡社区事务、农林水事务、交通运输等。这部分支出与基建投资高度相关，是财政“稳增长”的主要发力点。在经济下行期，我们会看到这部分支出的增速明显加快。
  * 维持运转类支出： 一般公共服务、国防、公共安全等。

* 赤字与平衡：
    一般公共预算赤字 = 一般公共预算支出 - 一般公共预算收入
  * 这个赤字，就是我们每年“两会”上最关注的“财政赤字率”（赤字/GDP）的分子。
  * 这个赤字如何弥补？主要通过发行国债和地方政府一般债券来融资。

### 政府性基金预算：“专款专用”的“副账户”

这是为支持特定公共事业发展而设立的、有专门收入来源和支出用途的“专款专用”账户。

* 收入来源：
  * 国有土地使用权出让收入（“土地财政”的来源，绝对大头）： 地方政府通过“卖地”获得的收入。这是这本账最重要的收入来源，其规模和增速，与房地产市场的景气度高度捆绑。
  * 其他：如彩票公益金收入、车辆通行费等。
* 支出方向：
  * 国有土地使用权出让收入相关支出： 主要用于征地拆迁补偿、土地开发、城市建设等。这是基建投资的另一大重要资金来源。
  * 其他专项支出。
* “以收定支”与专项债：
  * 政府性基金预算遵循“以收定支”的原则，理论上不列赤字。
  * 但近年来，随着“土地财政”收入的波动和基建投资需求的增加，出现了一个极其重要的创新工具——地方政府专项债券（专项债）。
  * 专项债，是一种有项目收益对应的、市场化发行的债券。它虽然在账面上被计入政府性基金的收入（作为“调入资金”），但其本质是一种债务融资，是“准赤字”。
  * 专项债的资金，被明确要求用于有一定收益的重大基建和项目投资，成为近年来驱动基建投资最核心的边际力量。

“两本账”的关系：广义财政的视角

在分析财政政策力度时，绝不能只看一般公共预算。必须将“两本账”结合起来，形成一个“广义财政”的视角。

* 广义财政赤字 = 一般公共预算赤字 + 地方政府专项债券新增额度
* 广义财政支出 = 一般公共预算支出 + 政府性基金预算支出

为什么要这样做？因为这两本账在功能上是互补的，有时甚至是“跷跷板”关系。

* 例如： 当房地产市场下行，土地出让收入大幅减少时，政府性基金的“钱袋子”就会瘪下去。为了对冲经济下行压力，继续支持基建投资，决策层往往会选择扩大一般公共预算的赤字规模，或者增加专项债的发行额度。
* 因此，广义财政赤字率（广义财政赤字/GDP），比狭义的财政赤字率，更能真实地反映当年积极财政政策的真实力度。

## 15.2 分析框架：从财政收支看政策力度与方向

掌握了“两本账”的基本构成，我们就可以构建一个分析财政政策的框架。这个框架，核心是回答两个问题：“政策的力度有多大？”（总量和节奏）和“政策在往哪里发力？”（结构）。

### 判断力度：总量与节奏

* 看目标（年初“两会”）：
    每年3月的《政府工作报告》和《预算报告》，是全年财政政策的“路线图”。我们需要关注以下几个关键目标数字：
  * 财政赤字率（狭义）： 目标是3%还是3.5%？赤字率的设定，直接反映了中央对当年财政扩张力度的定调。
  * 新增地方政府专项债券额度： 目标是3.6万亿还是3.8万亿？这个数字，直接决定了当年基建投资的“弹药”上限。
  * 税收收入/财政收入增速目标：
  * 财政支出增速目标：
    将这些目标与上一年的实际完成情况进行比较，我们就能对今年的财政政策是“更积极”还是“更收敛”，有一个总体的判断。

* 看执行（月度数据）：
    目标是静态的，执行是动态的。通过跟踪财政部每月公布的收支数据，我们可以判断政策的落地节奏。
  * 财政支出进度：
        支出进度 = 当月累计支出 / 全年目标支出
        如果上半年支出进度明显快于往年同期，这说明财政政策在“前置发力”，旨在“早落地、早见效”，对上半年经济的支撑作用会更强。
  * 专项债发行进度：
        同样，如果监管部门要求专项债在6月底或9月底前基本发行完毕，这也意味着基建投资的资金将在上半年或前三季度集中到位，形成“脉冲式”的拉动效应。
  * 留抵退税的节奏：
        近年来，大规模的“增值税留抵退税”成为“减税降费”的重要形式。它相当于直接给企业注入现金流。关注退税政策的规模和执行节奏，是判断财政对企业支持力度的重要方面。

* 看“超收”与“短收”：
  * 超收： 当经济景气度超预期时，税收收入可能会超过年初的目标，这会形成一笔“超收收入”。这笔钱可以用来削减赤字，也可以在年末“突击花钱”，形成额外的支出。
  * 短收： 当经济不及预期时，税收收入可能完不成目标，形成“短收”。此时，为了保证必要的支出，政府可能需要动用“预算稳定调节基金”（历年“超收”存下来的钱），或者在年中追加赤字/债务额度。

### 判断方向：结构是关键

财政支出的结构，比总量更能体现政策的意图。

* 支出结构的“跷跷板”：
    在财政总盘子相对固定的情况下，不同支出科目之间，存在此消彼长的关系。
  * “稳增长”压力大时： 我们会看到与基建相关的支出科目（如交通运输、城乡社区事务）增速显著加快。
  * 强调“调结构”、“惠民生”时： 我们会看到与科技、教育、社会保障、卫生健康相关的支出科目，增速会更快。
* 专项债投向的“风向标”：
    专项债的资金投向，是观察政策希望在哪些领域“补短板”、“促转型”的绝佳窗口。近年来，专项债的投向，从传统的“铁公基”，明显向产业园区、新型基础设施、新能源项目、保障性住房等领域倾斜。这反映了财政资金正在从“全面刺激”转向“精准滴灌”。
* 税收结构的“指挥棒”：
    除了支出，税收政策的结构性调整，也是引导经济行为的重要工具。
  * 研发费用加计扣除、高新技术企业所得税优惠： 鼓励企业进行科技创新。
  * 小微企业“六税两费”减免： 扶持小微企业，稳定就业。
  * 新能源汽车车辆购置税减免： 促进绿色消费和相关产业发展。

分析师的综合研判：

一个完整的财政分析报告，应该包含以下层次：

1. 总量判断： 今年广义财政的扩张力度，相比去年是强是弱？
2. 节奏判断： 财政发力的节奏是前置还是后置？对各季度的经济支撑作用如何？
3. 结构判断： 财政支出的重点，是在传统的“稳增长”领域，还是在代表未来的“调结构”领域，或是在托底民生的领域？
4. 风险评估： 当前的财政扩张，是否会带来地方政府债务风险的显著上升？财政的可持续性如何？

## 15.3 数据联动：赤字、国债与市场流动性

财政行为，并不仅仅作用于实体经济。它通过赤字的融资过程，与金融市场，特别是债券市场和银行间市场流动性，发生了深刻的联系。

### 赤字的弥补：国债与地方政府债券的发行

* 融资链条：
    财政赤字/专项债额度 → 债券发行需求 → 财政部/地方政府在银行间市场发行国债/地方债
* 谁来购买？
    这些政府债券的主要购买者，是商业银行。银行用其超额准备金（即在央行存放的、超出法定准备金要求的多余资金）来购买这些债券。
* 对市场流动性的影响：
    政府债券的大量发行，是一个从银行间市场“抽水”的过程。
  * 银行用自己的钱去购买了政府债券，其可以用于向其他银行融出资金或向实体放贷的“超额准备金”就减少了。
  * 这会导致银行间市场的流动性趋紧，推动市场利率（如DR007）上升。

### 央行的角色：是“对手方”，也是“护航人”

面对政府发债带来的“抽水”效应，央行通常会扮演“护航人”的角色，以维持银行间市场流动性的合理充裕。

* 联动逻辑：
    政府债券发行高峰期 → 银行间市场流动性紧张 → 央行通过公开市场操作（逆回购、MLF）或降准，向市场投放流动性，进行“对冲”。
* 分析师的应用：
  * 预判央行行为： 通过追踪每周的政府债券发行计划，我们可以预判未来一段时间银行间市场的资金面压力。如果预计下周将有大规模的国债和地方债发行，我们就可以合理地预期，央行大概率会通过加大逆回购投放等方式来“送温暖”，以平抑市场利率的波动。
  * 理解“降准”的动机之一： 央行实施“降准”，除了降低银行资金成本、鼓励其放贷之外，一个非常重要的目的，就是释放出长期、稳定的资金，让银行有足够的“弹药”去承接政府债券的发行。因此，我们常常看到，在地方债发行高峰来临之前，央行会择机进行降准。

### “财政赤字货币化”的争论

* 什么是财政赤字货币化？
    狭义上，指央行绕过商业银行，直接在一级市场上购买政府发行的债券（即央行直接印钱给政府花）。这是被《中国人民银行法》所明令禁止的。
* 广义上的联动：
    虽然央行不直接购买，但它在二级市场上，通过降准、MLF等工具为购买国债的商业银行提供流动性支持，客观上形成了“财政融资，央行买单（提供流动性）”的格局。这是一种间接的、市场化的协同配合。
* 与QE的区别：
    这与美联储等实施的“量化宽松”（QE）有本质区别。QE是央行直接在二级市场上大量购买国债等资产，其目的是压低长期利率，并直接向市场注入巨量流动性，央行的资产负债表会急剧扩张。而中国的模式，更多是“对冲”性质的，旨在维持流动性的平稳，而非“大水漫灌”。

### 财政存款的变化：对流动性的短期扰动

* 财政存款： 指政府的税收收入、债券发行收入等，在支出之前，会以“财政存款”的形式，存放在央行的国库中。
* 对流动性的影响：
  * 财政存款增加（通常在税收大月，如1、4、5、7、10月）： 相当于钱从商业银行体系，流向了央行的“保险柜”。这是一个“抽水”的过程，会收紧银行间市场的流动性。
  * 财政存款减少（通常在财政支出大月，如6、12月）： 相当于钱从央行的“保险柜”，通过财政支出，流回了商业银行体系。这是一个“放水”的过程，会增加银行间市场的流动性。
* 分析师的应用：
    通过预判每个月的财政收支节奏，我们可以对财政存款的变化，进而对当月的银行间市场流动性状况，做出一个大致的判断。比如，在季末财政支出高峰期，我们通常会看到资金面变得非常宽松。

## 本章小结

在本章，我们打开了政府的“钱袋子”，系统地学习了如何分析财政这只“看得见的手”。

我们厘清了一般公共预算和政府性基金预算这两本核心账本，并懂得了必须从“广义财政”的视角，来全面评估财政政策的真实力度。

我们构建了一个从总量、节奏和结构三个维度，去洞察财政政策力度与方向的分析框架。我们知道，要看懂财政，不仅要看年初的目标，更要追踪月度的执行；不仅要看总盘子，更要看钱花向了哪里。

最核心的，我们打通了财政与金融之间的联动。我们理解了赤字和国债发行如何从银行间市场“抽水”，以及央行又是如何通过各种工具为其“护航”的。我们还了解了财政存款这个短期变量，如何对市场流动性产生“扰动”。

掌握了财政分析，我们对宏观政策的理解，就从“知其然”达到了“知其所以然”的境界。在下一章，我们将去探索宏观调控的另一只巨手——货币政策。我们将走进央行的“工具箱”，看看利率、准备金率这些工具，是如何被用来调控经济的“血液循环”的。
