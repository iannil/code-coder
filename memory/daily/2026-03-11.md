### 安全保障

worker.ts 中的每个 SDK 调用都有 try-catch 回退机制：

```typescript
if (isSdkModeEnabled()) {
  try {
    // SDK 调用
  } catch (error) {
    Log.Default.warn("SDK session.xxx failed, falling back to local API", { error })
    // 回退到本地 API
  }
}
// 本地 API 调用
```

即使 Rust daemon 未运行或出现错误，TUI 仍能自动回退到本地 API 正常工作。
